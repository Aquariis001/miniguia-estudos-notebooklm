# 📚 Miniguia de Estudos com NotebookLM

> Projeto desenvolvido durante o desafio da DIO utilizando o NotebookLM como ferramenta de aprendizagem ativa baseada em Inteligência Artificial.

---

## 🎯 Objetivos

Este projeto teve como objetivo explorar os recursos do NotebookLM para organizar estudos sobre Inteligência Artificial Generativa, Modelos de Linguagem (LLMs) e Engenharia de Prompts.

Ao longo do desafio foram utilizados resumos, mapas mentais, tabelas comparativas e perguntas orientadas por IA para consolidar o aprendizado.

---
