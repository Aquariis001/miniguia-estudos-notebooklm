Além do aprendizado técnico, o desafio buscou desenvolver habilidades de:

* Curadoria de informações;
* Engenharia de Prompts;
* Organização do conhecimento;
* Pensamento crítico;
* Documentação do processo de aprendizagem.

# Miniguia de Estudos

## O que é IA Generativa?

É uma área da Inteligência Artificial capaz de produzir novos conteúdos, como:

* textos;
* imagens;
* músicas;
* códigos;
* vídeos.

Seu funcionamento é baseado principalmente em Grandes Modelos de Linguagem (LLMs).

---

## Modelos de Linguagem (LLMs)

Os LLMs aprendem padrões da linguagem por meio de enormes conjuntos de dados.

Seu funcionamento envolve:

* processamento por tokens;
* análise do contexto;
* predição do próximo token mais provável.

---

## O que são Tokens?

Tokens são pequenas unidades utilizadas pelo modelo para interpretar textos.

Podem representar:

* palavras;
* partes de palavras;
* caracteres.

---

## Importância do Contexto

O contexto permite que o modelo interprete corretamente o significado das palavras.

Exemplo:

A palavra **laranja** pode representar:

* uma fruta;
* uma cor.

O contexto define qual interpretação será utilizada.

---

## Engenharia de Prompts

Consiste em criar instruções claras e específicas para obter melhores respostas da IA.

Boas práticas:

* definir contexto;
* especificar formato da resposta;
* indicar público-alvo;
* limitar tamanho da resposta;
* solicitar exemplos.

---

## NotebookLM

O NotebookLM funciona como um parceiro de estudos baseado nas fontes fornecidas pelo usuário.

Entre seus recursos estão:

* resumos automáticos;
* mapas mentais;
* flashcards;
* podcasts;
* vídeos;
* quizzes;
* organização de documentos.

Seu diferencial é responder utilizando prioritariamente os documentos enviados pelo usuário.

# 📖 Glossário

**IA** — Inteligência Artificial.

**IA Generativa** — IA capaz de criar novos conteúdos.

**LLM** — Large Language Model.

**Token** — Unidade utilizada pelo modelo para processar texto.

**Prompt** — Instrução enviada ao modelo.

**Prompt Engineering** — Técnicas para construir prompts mais eficientes.

**Machine Learning** — Aprendizado de máquina baseado em dados.

**Deep Learning** — Aprendizado profundo utilizando redes neurais.

**Transformer** — Arquitetura utilizada pelos modelos modernos de linguagem.

**Grounding** — Técnica que limita as respostas às fontes fornecidas.

**RAG** — Recuperação de informações para gerar respostas mais precisas.

**Alucinação** — Resposta incorreta produzida pelo modelo.
